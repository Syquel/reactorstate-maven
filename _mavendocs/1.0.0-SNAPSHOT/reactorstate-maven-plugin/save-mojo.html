reactorstate:save

Full name:

de.syquel.maven.reactorstate:reactorstate-maven-plugin:1.0.0-SNAPSHOT:save

Description:

Saves the state of the current Maven execution for usage in subsequent Maven executions.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
  • Executes by direct invocation only.
  • Is NOT inherited by default in multi-project builds.

Parameters

(no parameters)
